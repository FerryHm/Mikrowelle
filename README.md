# Mikrowelle
Simuliert eine Mikrowelle.

## Anleitung
Ausschecken -> Starten

### Um die Mikrowelle zu bedienen:
Beim starten wird die Uhrzeit angezeigt. Drückt man + oder - wird automatisch in den Wattmodus geschaltet,
indem man die Watt einstellen kann. Drückt man <-> kommt man von den Wattmodus in den Zeitmodus und
umgekehrt. Im Zeitmodus kann man einstellen wie lange die Mikrowelle laufen soll. Die Zeit oder die
Wattanzahl wird durch + und - in 5er-Schritten angepasst. Durch Start startet die Mikrowelle. Wird nochmals
Start gedrückt wird die Zeit um 30 Sekunden erhöht. Derzeit gibt es noch keine automatische Aktualisierung
des Displays weshalb dieses manuell mithilfe von Update gemacht werden muss. Desweiteren hat Open noch
keine Funktion.

Mithilfe von 1, 2, 3 und 4 lassen sich Presets starten, das sind voreingestellte Modi, die direkt starten.

Stellt man Watt und Zeit ein und geht dann auf ein Preset, wird die Einstellung dafür übernommen.
